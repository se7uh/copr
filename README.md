# My COPR Repository

RPM spec files for software packages I maintain on [Fedora COPR](https://copr.fedorainfracloud.org).

---

## Packages

| Package | Description | Upstream | COPR |
|---------|-------------|----------|------|
| bustd   | Lightweight OOM-killer daemon | [vrmiguel/bustd](https://github.com/vrmiguel/bustd) v0.1.1 | [boobaa/bustd](https://copr.fedorainfracloud.org/coprs/boobaa/bustd/) |
| ouch    | Painless compression in the terminal | [ouch-org/ouch](https://github.com/ouch-org/ouch) v0.8.0 | [boobaa/ouch](https://copr.fedorainfracloud.org/coprs/boobaa/ouch/) |

---

## Usage

```bash
sudo dnf copr enable boobaa/<package>
sudo dnf install <package>
```
